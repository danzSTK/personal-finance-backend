# Archived Account

Uma account arquivada deixa de aparecer na listagem padrão.

## Regras Atuais

- Contas arquivadas só aparecem quando o usuário pede explicitamente.
- Contas arquivadas não devem aceitar alterações comuns.
- Uma account arquivada não pode virar default.

## Uso

Arquivar preserva histórico sem remover a account do sistema.
