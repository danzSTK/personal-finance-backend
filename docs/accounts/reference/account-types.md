# Account Types

| Tipo | Status | Uso |
|---|---|---|
| `CASH` | planned V0 | Dinheiro físico/default estrutural criado no onboarding |
| `BANK` | current | Conta bancária comum |
| `CREDIT_CARD` | current enum | Tipo previsto, sem regra profunda na V0 |
| `INVESTMENT` | current enum | Tipo previsto, sem regra profunda na V0 |

Na V0, o produto deve operar principalmente com `CASH` e `BANK`.
